How to Save These Project Images

  1. Take a screenshot of each project card below (you can use Windows+Shift+S on Windows or Command+Shift+4 on Mac).
  2. Save each screenshot with the correct filename in the public/projects/ directory:
    • educadamy.jpg - for the purple EduCadamy card
    • pufftea.jpg - for the green Puff Tea card
    • portfolio.jpg - for the orange Portfolio card
  3. Make sure the images are saved in JPG format.
  4. After saving all three images, refresh your portfolio page to see them displayed.
